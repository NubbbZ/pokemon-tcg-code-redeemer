# Pokemon TCG Code Scanner
### Pokemon TCG Code Scanner aims to make scanning pokemon codes a breeze!

This project is minimal and easy to run, You can use it on any device with a web-browser!

Available at: https://nubbbz.github.io/pokemon-tcg-code-scanner/

> This project is a fork of the original available here: https://jordanbird.github.io/pokemon-tcg-code-redeemer/
